# CEH Eğitimi - Siber Güvenlik ve Etik Hackerlik

**Eğitmen:** Halil Yaşarsoy

## Eğitim Hakkında

Bu eğitim, askeri personel ve siber güvenlik profesyonelleri için hazırlanmış kapsamlı bir Certified Ethical Hacker (CEH) eğitim programıdır. Eğitim, siber güvenlik temellerinden ileri seviye sızma testi tekniklerine kadar geniş bir yelpazede konuları kapsamaktadır.
